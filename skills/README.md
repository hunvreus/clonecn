# Skills

Canonical, publishable skills live in this directory.

- `clonecn/`

## Ship via GitHub

Install directly from this repo:

```bash
python3 "${CODEX_HOME:-$HOME/.codex}/skills/.system/skill-installer/scripts/install-skill-from-github.py" \
  --repo <owner>/clonecn \
  --path skills/clonecn
```

Then restart Codex and invoke with `$clonecn`.
